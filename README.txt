This is a school project made by 4 students. The music subscription app lets you send data to the local or any server via SSL connection.
When you connect to a server you can complete the form and send it securely via SSL to the server. There is also a logger 
where you can type commands and communicate with the server.
List of commands:
help - shows a list of commands in a logger
self connect - connects to the local server
connect <ip> <port> -connects to the external server
disconnect - disconnects from the server
